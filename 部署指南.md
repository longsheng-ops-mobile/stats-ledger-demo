# 🚀 部署到 GitHub Pages 操作指南

## 只需3步，让任何人通过互联网访问您的Demo

---

### 第1步：创建GitHub仓库

1. 打开浏览器，访问 **https://github.com/new**
2. 登录GitHub账号（如果没有，先注册一个：https://github.com/signup）
3. 填写仓库信息：
   - **Repository name**: `stats-ledger-demo`（或任意英文名）
   - **Description**: `统计局台账数据采集与汇总系统`
   - 选择 **Public**（公开，GitHub Pages免费版要求公开仓库）
   - **不要**勾选 "Add a README file"
4. 点击 **Create repository**

---

### 第2步：推送代码

创建完仓库后，页面会显示仓库地址。打开终端，执行以下命令：

```bash
cd /Users/longsheng/WorkBuddy/20260326111649/统计局台账系统/deploy

# 替换为您的GitHub用户名
git remote add origin https://github.com/您的用户名/stats-ledger-demo.git

# 推送代码
git branch -M main
git push -u origin main
```

推送时会弹出GitHub登录窗口，输入您的GitHub账号密码或Token即可。

---

### 第3步：开启GitHub Pages

1. 打开刚创建的仓库页面：`https://github.com/您的用户名/stats-ledger-demo`
2. 点击顶部的 **Settings**（设置）
3. 左侧菜单找到 **Pages**
4. 在 "Build and deployment" 下：
   - **Source** 选择 `Deploy from a branch`
   - **Branch** 选择 `main`，文件夹选 `/ (root)`
5. 点击 **Save**

---

### ✅ 完成！

等待约1-2分钟，您的Demo就可以通过以下地址访问了：

**https://您的用户名.github.io/stats-ledger-demo/**

把这个链接发给任何人，他们打开浏览器就能看到完整的演示！
